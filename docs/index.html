Wissensbilanz Dashboard

Ein Promptotyping-Lehrbeispiel für die VetMed Wien

Was ist Promptotyping?

Promptotyping ist eine Methode zur schnellen, iterativen Entwicklung von Software-Prototypen mit KI-Unterstützung. Dieses Dashboard demonstriert den Prozess.

Build-Informationen

Version: 1.0.0-beta
Build-Datum: 2026-01-30
Technologie: Vanilla JS + ES6 Modules
Charts: Chart.js 4.4.1
Datenquelle: unidata.gv.at (2021-2024)

Architektur

  • EventBus: Entkoppelte Komponenten-Kommunikation
  • State: Zentraler, reaktiver Zustand
  • Router: URL-Synchronisation für teilbare Links
  • VizFactory: Austauschbare Visualisierungen
